# 数理計画とアルゴリズム report3
「数理計画とアルゴリズム」 report3の課題プログラムです.

## Description
大学の講義のプログラム「数理計画とアルゴリズム」のreport3の課題プログラムです.
課題は次のようになっています.
- 練習課題をシンプレクス法で解くプログラムを作成せよ.<br>
  「simplex_method.py」


## Dependencies
- [Python3](https://www.python.org/)
- [NumPy](http://www.numpy.org/)
- [pandas](http://pandas.pydata.org/)
